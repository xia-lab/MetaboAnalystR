Manually select a subset of features/samples for ROC analysis

The list boxes below show all the available features and samples. You can manually select the features you want to use to build a classifier or you can select a subset of samples as hold-out data for validation purposes. Note, multiple selections are allowed.

You should select features based on your own judgement or prior knowledge. If you select features based on their overall ranks (AUC, T-statistic or fold changes), there will be an increase risk of overfitting - they may be the best biomarkers for this particular data, but not the case for new samples! The performance evaluation will likely to be over-optimistic. Use with caution!. You can use K-means (KM) clustering to detect features with similar behaviour to help reduce the redundancy in biomarkers (i.e. features in the same cluster behave more similar).

Note, directly click on the checkbox in the 1st column for multiple selection. Do not click on other places of the row.
