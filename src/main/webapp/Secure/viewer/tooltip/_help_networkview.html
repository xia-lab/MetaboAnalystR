• Zoom: use your mouse scroll / swipe trackpad;
  • Hover over a node to see its label and all its connections;
  • Hover over an edge and click on it to show the evidence for the corresponding interactions;
  • Highlight node(s): by double clicking a node (see Scope above); by enter a list of genes for Batch selection; or by clicking an enriched function or module;
  • Scope: this option will affect nodes included in drag-and-drop. The first two options will also affect nodes being highlighted when double clicking an individual node;
  • Drag-and-drop: hover on a node, mouse down to drag the node to a new position (see Scope above). Note, you can manually select and drag multiple nodes using the Marquee icon on the tool bar.
  • View options: allow you to further customize views for Nodes, Edges, and Highlighting effect.
