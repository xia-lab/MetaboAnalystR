MSP Spectra Integrity Check:

  1. MetaboAnalyst-Pro only supports tandem spectra searching on level 2. Other levels will be excluded.
  2. Compared to public version, MetaboAnalyst-Pro supports unlimited spectra in an individual search.
  3. User could manually edit the precursor ion inclusion list for MS/MS annotation, unless the total number of MS/MS spectra in your msp file is < 20.
  4. Searching < 20 MS/MS will be processed immediately. The searching job (of over 20) will be scheduled to be performed once you click Submit and Confirm button.
