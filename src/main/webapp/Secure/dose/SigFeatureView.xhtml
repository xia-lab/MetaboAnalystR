Identification of potential features associated with exposure levels

Two general approaches are implemented. The downstream curve fitting step will be performed on those significant features in the next step

  • Differential analysis: to identify features that have shown significant differences / variations in ANY of the dose treatment with respective the control (dose 0). This is based on a general ANOVA-like tests using limma (repeated dosing data).
  • Regression analysis: to identify features that have shown significant association with the dose levels. This is based on logistic (two groups) or linear regression (continuous exposure data)

Did you know?

For dose response analysis, this step is to select features that are more likely to have a dose-dependent behavior using fold-change and/or p-values. Since these features will be subject to additional filters during the curve-fitting step, you can use relaxed thresholds. However, if there is a strong effect, we recommend increasing the threshold values such that fewer than 1000 features pass this step for computational reasons.

Please note: a feature will be selected if its fold-change exceeds the threshold in any of the pairwise comparisons.

Did you know?

For dose response analysis, this step is to select features that are more likely to have a dose dependent behavior using correlation coefficient (r) and/or p-values. Since these features will be subject to additional filters during the curve-fitting step, you can use relaxed thresholds. However, if there is a strong effect, we recommend increasing the threshold values such that less than 1000 features pass this step for computational reasons.
